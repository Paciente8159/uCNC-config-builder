µCNC config generator

NOTICE

Web Builder might not always generate 100% correct config files.

Sometimes the generated config files, cause compilation errors and sometimes the code just compiles fine but does not work due to a bad configuration.

ALWAYS CHECK THE GENERATED FILES:

  • make sure to inspect the gererated override files to see if all your settings were reflected:
    • all your new settings are defined after the #undef declaration
    • all old settings deleted have one #undef declaration
  • make sure you have all your necessary pins defined
  • you have downloaded the correct modules version files for the version you are using
  • clean/undefine unecessary or unused pins to prevent unwanted issues
  • not all configuration might be available in the web builder. For custom boards and MCU additional configurations might be needed

Issues

If you encounter any issue please consider submiting an issue in the Github project page, with an attached zip with all the override files and the json config file.

Config µCNC

To build your custom version of µCNC follow these steps:

  • Download and unzip µCNC ({{getVersion(VERSION)}})
  • µCNC addon modulesIf you are using modules you need to download them place each used module directory inside uCNC/src/modules/ directory
  • Download boardmap_overrides.h And replace the existing file inside uCNC root directory
  • Download cnc_hal_overrides.h And replace the existing file inside uCNC root directory
  • Compile and upload the project either via Platform IO (recommended) or Arduino IDE.
  • Download config settings These can be later used to reload your settings
